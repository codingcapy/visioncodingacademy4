ENTRY LEVEL

G1 - G3

Overview:

Embark on "Begin with Bytes," a captivating 24-week adventure tailored for Grades 1-3. This course is specially designed to introduce the youngest learners to the wonders of coding and technology. Through playful, interactive experiences, we aim to spark curiosity and lay the groundwork for digital fluency.

Weeks 1-2: Introduction to Computers

  • Topics:
    • What is a Computer?
    • Parts of a Computer
    • Using a Computer Safely.
  • Activity:
    • Drawing or crafting a computer and its parts.
    • "Computer Hunt" - Find computer chips in the classroom.

Weeks 3-4: Discovering the Internet

  • Topics:
    • What is the Internet?
    • Understanding Websites.
  • Activity:
    • Navigating kid-friendly websites, creating a storyboard of how they connect to the world.
    • "Safe Surfing" - Explore the internet safely

Weeks 5-6: Introduction to Block-Based Coding with ScratchJr

  • Topics:
    • Basics of ScratchJr
    • Creating Simple Animations.
    • Read and understand scratch documentation
  • Project:
    • Create a simple animated story or game.

Weeks 7-8: ScratchJr - Part 1

  • Topics:
    • Creating Interactive Stories in ScratchJr.
    • Learn different functions and features within Scratch Jr.
  • Project:
    • Students create their own interactive story.

Weeks 9-10: ScratchJr - Part 2

  • Topics:
    • Loops and Repetition
  • Activity:
    • ​"Loop the Loop" - A hands-on activity to create looping animations or patterns in ScratchJr.

Weeks 11-12: ScratchJr - Part 3

  • Topics:
    • Conditional Statements
    • Introduction to Variables
  • Activity:
    • "Interactive Quiz" - Students build a simple quiz that reacts to user choices.
    • "Scorekeeper" - Create a game in ScratchJr that keeps track of scores using variables.

Weeks 13-14: ScratchJr - Part 4

  • Topics:
    • Create final project with ScratchJr
  • Activity:
    • "Mini Coding Fair" - Students work on a small project integrating the concepts learned and present it to the class.

Weeks 15-16: Introduction to Digital Art

  • Topics:
    • What is Digital Art?
    • Creating Art on a Computer​
  • Project:
    • "Digital Picasso" - Use ScratchJr to create digital art pieces.

Weeks 17-18: Understanding Algorithms

  • Topics:
    • Understanding Patterns, Creating Sequences.
    • Basics of Algorithms​
  • Activity:
    • "Treasure Hunt Algorithm" - Develop a step-by-step guide to find a hidden object in the classroom.

Weeks 19-20: Project-Based Learning and Application

  • Topics:
    • Collaborative Digital Storytelling
    • Learn to Collaborate and Code together​
  • Project:
    • "​Group Story Project" - Students form groups to create and code an interactive story.

Weeks 21-22: Group Project: Simple Game Development

  • Topics:
    • Basic Game Development
    • Understand the process to create game​
  • Project:
    • "Create a basic game (e.g., a simple maze or catching game).
    • "Classroom Arcade" - Groups design and develop simple games, then playtest each other's creations.

Weeks 23-24: Showcase and Review

  • Topics:
    • Review of Concepts from previous weeks
    • Preparing for Showcase​
  • Activity:
    • "Coding Showcase" - Finalize projects and prepare presentations for a class coding fair.
