ENTRY LEVEL

G4 - G10

Overview:

Welcome to "Discover Coding," a 24-week journey designed to introduce young minds to the fascinating world of coding and computer science. This course is tailored for students aged G4-G10, providing a nurturing and engaging environment to explore the building blocks of technology.

Weeks 1-2: Introduction to Computers and Basic Programming Concepts

  • Topics:
    • History of Computers
    • Basic Computer Operations
    • What is Programming?
  • Activity:
    • Create a timeline of computer history or a basic flowchart for a daily routine.

Weeks 3-4: Introduction to Scratch

  • Topics:
    • Scratch interface
    • Basic programming blocks
    • Creating simple animations
  • Activity:
    • Create a simple animation, like a dancing character or a short story.

Weeks 5-6: Advanced Scratch Projects

  • Topics:
    • Advanced Scratch features
    • Create some complex games
    • Introduction to logic.
  • Project:
    • Design a game (e.g., maze, simple platformer) in Scratch.

Weeks 7-9: Introduction to Python

  • Topics:
    • Python setup
    • Learn basic Syntax, Variables, Basic Data Types
    • Learn VS Code
  • Project:
    • Write basic Python scripts (ex: calculates the area of shapes.)

Weeks 10-12: Control Structures and Basic Data Structures in Python

  • Topics:
    • Conditional Statements, Loops, Lists, Tuples.
    • Understand how to use terminal in VS Code
  • Project:
    • Develop a Python program for a simple application a to-do list.

Weeks 13-15: Functions, Modules, and File Handling in Python

  • Topics:
    • Defining Functions, Importing Modules, File I/O.
    • Learn basics of automation
  • Project:
    • Create a Python script that reads a file (e.g., a text log) and summarizes its content.
    • Automate some kind of daily tasks

Weeks 16-17: Introduction to Web Development: HTML & CSS

  • Topics:
    • HTML tags
    • CSS styling
    • Creating a static webpage
  • Project:
    • Build a personal or hobby-related webpage using HTML and CSS.
    • Use VS extensions (live, prettify)

Weeks 18-19: Introduction to JavaScript and Interactive Web Pages

  • Topics:
    • JavaScript syntax, DOM manipulation, Responding to user inputs.
  • Project:
    • Add interactive features to the static webpage (e.g., a form, dynamic content) to the webpage.

Weeks 20-21: Introduction to Backend Development with Node.js

  • Topics:
    • Basics of Node.js
    • Setting up a local server​
    • Basic routing (8080 / local host)​
  • Project:
    • Develop a simple web server that serves your HTML page.
    • Make your web go alive and share with your friends.

Weeks 22-23: Full-Stack Mini-Project

  • Topics:
    • Integrate front-end with back-end.
    • Basic full-stack application development.​
  • Project:
    • Create a basic full-stack web application; a blog or a personal portfolio site.

Week 24: Course Recap and Showcase

  • Topics:
    • Review of key concepts
    • Project presentations.​
  • Activity:
    • Students present their projects, sharing their learning experiences and receiving feedback.
